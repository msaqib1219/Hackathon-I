# Week 3: Reasoning Patterns

## Why this matters
LLMs can make mistakes if they rush to an answer. Forcing them to break down problems improves accuracy significantly.

## Core Concepts
- **Chain of Thought (CoT)**: Step-by-step reasoning.
- **Self-Consistency**: Generating multiple thoughts and voting.

## Teaching the agent to "think out loud"
By exposing the internal monologue, we can debug the agent's logic.
